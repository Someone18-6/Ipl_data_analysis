# IPL Data Analysis Project

This project analyzes team performance in the Indian Premier League using Excel/Google Sheets.

## Features
- Win percentage calculation
- Matches played tracking
- Team ranking system
- Conditional formatting for insights
- Interactive charts

## Tools Used
- Excel / Google Sheets
- Basic data analytics

## Key Insights
- Teams ranked based on Win %
- Tie-breakers handled using matches played
- Visual dashboard for performance comparison
